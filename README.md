# Python Fundamentals Portfolio

A curated collection of Python programs demonstrating core computer science concepts such as algorithms, recursion, problem-solving, and basic game logic.

## Contents
- **algorithms/** — sorting & searching (bubble sort, insertion sort, linear/binary search)
- **recursion/** — recursion practice (e.g., factorial)
- **games/** — small games (e.g., Hangman, Tic-Tac-Toe, Guess the Number)
- **problem_solving/** — logic/conversion problems (e.g., roman numerals, binary to decimal)
- **practical_apps/** — small utility programs (e.g., calculator, password generator)

## How to run
Most files can be run with:
```bash
python filename.py
